# Recipe: 科普/解说(1–10min,横屏为主)

结构化讲清一件事:章节化 + 图示 + 数据卡。

- **开头 10s**:这期讲什么 + 为什么值得看完(B站/YouTube 的完播关键)。
- **章节化**:每 60–90s 一个小节,小节标题全屏过场(观众心理进度条);YouTube 描述里给时间戳。
- **视觉节奏**:纯口播画面 ≤ 15s 必须切视觉(图示/数据卡/引用卡);数据必须可视化(用模板的数据卡组件),不许口播念数字了事。
- **字幕**:横屏 ≤ 24 字/行;术语首次出现时字幕高亮 + 停留略长。
- 起手复制 `templates/kepu-horizontal.html`。
